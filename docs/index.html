🛡️ Guardian

Free lightweight EDR for teams who can't afford commercial endpoint detection

⚡ One command to start monitoring

Python agent that watches processes, files, and network. Web dashboard included. No cloud, no subscription, no data leaving your network.

pip install psutil flask pyyaml && wget -O guardian.py https://git.io/guardian-edr && python guardian.py
🔍

Process Monitoring

Detects new processes, flags suspicious tooling in real-time. PowerShell encoded commands, Mimikatz, BloodHound, and more.

📁

File Monitoring

Watches temp directories and sensitive paths for new files created by attackers during post-exploitation.

🌐

Network Visibility

Tracks active outbound connections. See what processes are phoning home to C2 infrastructure.

📊

Web Dashboard

Real-time alert dashboard in your browser. Filter by severity, inspect details, export findings.

🔍 Built-in detection rules

Mimikatz BloodHound CrackMapExec PowerShell -enc PowerShell IEX AMSI Bypass curl/wget downloads certutil.exe bitsadmin New files in /tmp Process exits

Add custom YAML rules in detector/rules/ — no code changes needed.

📖 Usage

# Run with dashboard
python guardian.py

# Run headless (no web UI, for servers)
python guardian.py --headless

# Specify port
python guardian.py --port 8080

# Run for 60 seconds then exit (test mode)
python guardian.py --duration 60

🚨 When to use Guardian

  • You're a solo consultant or small MSSP — can't justify $10k/yr for EDR seats
  • You need quick triage on a可疑 host — run Guardian, check the dashboard, find anomalies
  • You're learning incident response — Guardian shows you what EDR telemetry looks like
  • You want to monitor a lab environment — no license, no restrictions

📦 Requirements

  • Python 3.8+
  • Linux or Windows
  • psutil for process/network monitoring
  • flask for web dashboard
  • No internet required — everything runs locally
